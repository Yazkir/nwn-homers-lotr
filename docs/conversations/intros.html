We have fixed death so that you cannot exit the game and come back fully recovered. This is done using a death amulet placed on the corpse at the time you expire. In most instances this amulet is destroyed once you have been properly revived. However if you are ressurected by another player the Death Amulet will still be on your character. Drop it immediately. Otherwise the next time you log on you will be dead again.
